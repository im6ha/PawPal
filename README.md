#  PawPal

**PawPal** is a web platform designed to connect pet lovers — allowing users to adopt pets, find sitters, report lost animals, and access pet care resources.

---

##  Website Structure

###  Main Pages
- **Home Page** – Overview of the platform and navigation to all features.
- **Adoption Page** – Browse and adopt pets.
- **Market Page** – Buy or sell pet-related products and services.
- **Find Sitters Page** – Connect with verified pet sitters nearby.
- **Lost Pets Page** – Report and find missing pets.
- **Pet Care Guide** – Articles and advice for pet care.



##  Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP-Laravel
- **Database:** MySQL
- **Version Control:** Git + GitHub
- **Collaboration:** GitHub Organization (ENSIA-AI)
